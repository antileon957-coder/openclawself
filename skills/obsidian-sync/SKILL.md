# Obsidian Sync Skill

Use this skill to sync files from OpenClaw workspace to Obsidian vault.

## Usage

```bash
./google-search-env/bin/python3 skills/obsidian_sync.py <source_file_or_dir> <category>
```

- `source_file_or_dir`: The file or directory in `artifacts/` to sync.
- `category`: The subfolder name in Obsidian Vault (e.g., "Agent_Outputs", "Research").

## Configuration

- Target Vault: `/Users/tayloryang/Documents/Obsidian Vault`
